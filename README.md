### icon库

**使用iconfont:**

- 将所需图标加入项目

- 下载字体文件到本地

![](https://s1.328888.xyz/2022/08/20/B2WDF.png)

- 将文件覆盖至项目中font目录

![](https://s1.328888.xyz/2022/08/20/B2d1S.png)

- 生成最新在线路径覆盖`font/iconfont.css`中的`@font-face{}`

![](https://s1.328888.xyz/2022/08/20/BIEvN.png)

- 使用示例

```ht
<i class="iconfont icon-sousuo"></i>
```
